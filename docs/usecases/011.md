# Use Case 11: Delete Comment on Recipe

**Use Case Name:** Delete Comment on Recipe

**Description:** This use case describes how an administrator deletes a comment on a recipe.

**Actors:** 
- Administrator

**Preconditions:** 
- The administrator is logged in with admin privileges.
- The comment exists on the recipe.
- The administrator has access to the internet and the web application.

**Main Flow:**
1. The administrator navigates to the recipe details page where the comment is located.
2. The administrator selects the option to delete the comment.
3. The system prompts the administrator to confirm the deletion.
4. The administrator confirms the deletion.
5. The system deletes the comment from the database.
6. The system removes the comment from the recipe details page.

**Alternative Flows:**
- 4a. If the administrator cancels the deletion:
  - 4a1. The system aborts the deletion process and retains the comment.

**Postconditions:**
- The comment is removed from the database.
- The comment no longer appears under the recipe details.

**Special Requirements:**
- The system should allow only administrators to delete comments.
