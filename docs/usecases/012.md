# Use Case 12: Manage Ingredients

**Use Case Name:** Manage Ingredients

**Description:** This use case describes how an administrator manages ingredient suggestions used by the application, including searching, creating, editing, and deleting ingredients.

**Actors:**
- Administrator

**Preconditions:**
- The administrator is logged in with admin privileges.
- The administrator has access to the internet and the web application.

**Main Flow:**
1. The administrator navigates to the ingredient administration page.
2. The system displays the ingredient management interface with search and editing options.
3. The administrator searches for an existing ingredient by name or starts creating a new ingredient.
4. If an existing ingredient is selected, the system loads its current details.
5. The administrator enters or updates the ingredient name and allergen information.
6. The administrator submits the changes.
7. The system validates the data.
8. The system saves the new ingredient or updates the existing ingredient in the database.
9. The system displays a success message and refreshes the search results.

**Alternative Flows:**
- 3a. If the administrator enters an empty search term:
  - 3a1. The system does not perform the search and displays a hint to enter at least one character.
- 6a. If the ingredient name is empty:
  - 6a1. The system displays an error message indicating that the ingredient name is required.
- 8a. If the administrator chooses to delete an existing ingredient:
  - 8a1. The administrator selects the delete option.
  - 8a2. The system deletes the ingredient from the database.
  - 8a3. The system displays a confirmation message and clears the form.

**Postconditions:**
- The ingredient suggestion list is updated in the database.
- The administrator can immediately search for and view the changed data.

**Special Requirements:**
- Only administrators may access ingredient management functions.
- The system should allow allergen information to be assigned to ingredients.
- The interface should support quick search and editing for existing ingredient suggestions.
