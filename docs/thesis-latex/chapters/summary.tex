\chapter{Összegzés  }\label{ch:summary}

Összességében sikerült megvalósítanom a kitűzött főbb követelményeket. Először dolgoztam ASP.NET Core rendszerrel, így a backend fejlesztése különösen tanulságos volt számomra. Sokat fejlődött a generikus kód írási képességem is. A szigorú alapkövetelményeknek köszönhetően könnyedén tudtam bővíteni a rendszert funkciókkal.

Legfontosabb célkitőzésem az volt, hogy a szoftver nagyon könnyedén indítható legyen helyi környezetben, minimális külső függőséggel. A szakdolgozat írása közben ismerkedtem meg jobban a GitHub, GitHub Actions és a Docker használatával. Ennek köszönhetően a kód felhőben tárolva, biztonságosan verziókezelve van. A GitHub Actions segítségével pedig egy teljesen automatizált build folyamatot hoztam létre.


\section{További fejlesztési lehetőségek}

A webalkalmazás jelenlegi állapotában is teljes funkcionalitást nyújt, azonban számos lehetőség van a további fejlesztésre. Néhány lehetséges irány:
\begin{itemize}
    \item email szolgáltatás integrálása a regisztráció és a jelszóvisszaállítás folyamatába
    \item közösségi média integráció, például Google vagy Facebook bejelentkezés
    \item címkézés és kategorizálás a receptekhez, hogy könnyebben lehessen keresni és szűrni őket
    \item receptajánló rendszer fejlesztése, amely a felhasználói preferenciák és korábbi aktivitás alapján ajánl recepteket
    \item mobilalkalmazás fejlesztése a webalkalmazás mellett
    \item keresési és szűrési funkciók továbbfejlesztése, például összetettebb szűrők bevezetése
    \item unit és integrációs tesztek bevezetése, elsősorban az egyes komponensek finomabb szintű ellenőrzése érdekében
\end{itemize}


\section{Köszönetnyilvánítás}

Szeretnék köszönetet nyilvánítani Kovács Sándor Márknak a folyamatos támogatásért, a hasznos tanácsaiért és a szakmai útmutatásért. Sok időmbe telt, mire megírtam ezt a dolgozatot, és a segítsége nélkül nem lett volna lehetséges. Köszönöm a türelmét és a biztatását páromnak, volt csoportársaimnak és barátaimnak, akik végig mellettem álltak, és segítettek megtartani a motivációmat.
A dolgozat elkészítése során szerzett tapasztalatok jelentősen hozzájárultak a szakmai fejlődésemhez.
